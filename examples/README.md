# Open UDP — Exemplos

Esta pasta destina-se a exemplos de implementação e uso do protocolo Open UDP (`usp.delivery`).

## Estrutura planejada

- **basic/** — Fluxo mínimo: BROADCAST_ORDER → SUBMIT_BID → AWARD_ORDER → UPDATE_STATUS (DELIVERED).
- **advanced/** — Risk Premium, REPORT_INCIDENT, Proof of Delivery, integração com relay/broker.
